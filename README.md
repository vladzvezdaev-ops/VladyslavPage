# Vladyslav Workspace 🌌

Interactive digital portfolio environment with a focus on high-end aesthetics and fluid animations.

## ✨ Features
* **Liquid Glass UI:** Custom glassmorphism components with iOS-style saturation and blur.
* **Interactive Canvas:** Dynamic particle system background that reacts to movement.
* **Advanced Motion:** Powered by Framer Motion for smooth, physics-based interactions.
* **Tech Stack:** React 19, Vite, TypeScript, and SCSS Modules.
